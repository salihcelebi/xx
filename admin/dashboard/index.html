Yönetim Paneli Tam Türkçe

Dosya Bağımlılıkları

  • index.html → static/kame-ui.js + index.css + puter.js
  • src/app.js → router + store + ui shell
  • src/router.js → pages/* render modülleri
  • src/services/* → puter.ai/usage/auth köprüsü

Paket / Özellik Matrisi

  • Free: temel chat + sınırlı model
  • Pro: video/görsel/ses + gelişmiş model havuzu
  • Enterprise: tüm tool'lar + admin raporlama + denetim logları

Operasyon Notları

  • Tool uyumsuzlukları route değişiminde otomatik kapanır.
  • Mod bazlı taslaklar localStorage üzerinde tutulur.
  • Provider değişiminde kurumsal renk teması otomatik uygulanır.
